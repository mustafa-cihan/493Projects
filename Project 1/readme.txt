
Program is written in Python 3.10.1

Program is run by running the solution.py file:

    "python3 solution.py"

reuters21578 must be in the same directory as solution.py in order to create the reverse index.

The program will then ask for a query. The query can be in any case and can contain punctuation.

Phrases can be searched by enclosing the phrase in quotes. ex: "new york"

Proximity searches can be done by adding word1 number word2 without quotes. Word must not contain whitespace. ex: new 3 york

To exit the program, type "exit" without quotes.
